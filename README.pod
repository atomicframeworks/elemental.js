<h1> <a href="http://www.atomicframeworks.com/elemental">Elemental.js </a> - The jQuery Element Factory </h1>
Elemental.js was created to ease building, styling, and binding elements on the fly.
<br />
<h1>Installing</h1>
Installing Elemental is easy.  Simply download it from GitHub and include the elemental.js file.
<h1>Requires</h1>
Elemental.js requires the jQuery library.
<h1>Features</h1>
<blockquote>
<strong>Easily set HTML Attributes:</strong> Passing in an 'html' object will allow for creating html attributes during element construction
<br />
<br />
<strong>Easily Set CSS Attributes:</strong> Just pass a 'css' object attrbutes of CSS attributes to set
<br />
<br />
<strong>Easily Set Binds and Callbacks:</strong>  Pass an 'on' object with bundled callbacks to bind & namespace them
</blockquote>
